# BAMTIBAM- 개발자용 쇼핑몰 

VS Code 스타일의 네온 테마를 적용한 React 기반 쇼핑몰 웹사이트입니다.
**주의: 보안이 거의 없는 **

## 주요 기능

- **투박한 로그인/회원가입**: 검증 거의 없는 사용자 인증 시스템 (새 창 팝업)
- **카테고리별 상품 조회**: AI, 건강, 전자기기, 언어, 스트레스 카테고리
- **상품 상세 모달**: 상품 정보, 리뷰, 구매 기능
- **관리자 페이지**: 회원 관리, 상품 관리, 리뷰 관리, 게시글 관리
- **투박한 디자인**: 학생이 만든 것처럼 간단하고 투박한 UI

## 디자인 특징

- **VS Code 테마**: 검정 배경에 네온 포인트 컬러
- **네온 효과**: 글로우 효과와 애니메이션
- **코딩 스타일**: 개발자를 위한 UI/UX

## 기술 스택

- **React 18**: 프론트엔드 프레임워크
- **React Router**: 페이지 라우팅
- **Vite**: 빌드 도구
- **CSS3**: 커스텀 스타일링

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 브라우저에서 확인
- http://localhost:3000 접속

## 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── Layout.jsx      # 메인 레이아웃
│   ├── Layout.css      # 레이아웃 스타일
│   ├── ProductModal.jsx # 상품 상세 모달
│   └── ProductModal.css # 모달 스타일
├── pages/              # 페이지 컴포넌트
│   ├── Home.jsx        # 메인 페이지
│   ├── Home.css        # 홈 스타일
│   ├── Login.jsx       # 로그인 페이지
│   ├── Signup.jsx      # 회원가입 페이지
│   ├── Auth.css        # 인증 페이지 스타일
│   ├── Admin.jsx       # 관리자 페이지
│   └── Admin.css       # 관리자 스타일
├── App.jsx             # 메인 앱 컴포넌트
├── App.css             # 앱 전역 스타일
├── main.jsx            # 앱 진입점
└── index.css           # 전역 스타일
```

## 주요 페이지!!

### 1. 홈페이지 (비로그인)
- 웹사이트 소개
- 주요 기능 안내
- 로그인/회원가입 링크

### 2. 메인페이지 (로그인 후)
- 카테고리별 상품 목록
- 상품 카드 클릭 시 상세 모달
- 사이드바 카테고리 네비게이션

### 3. 로그인/회원가입
- 폼 유효성 검사
- 로그인 성공 시 메인페이지 이동
- 테스트 계정 제공

### 4. 관리자 페이지
- **회원 관리**: 사용자 차단/해제
- **상품 관리**: 상품 활성화/비활성화
- **리뷰 관리**: 리뷰 삭제, 신고 관리
- **게시글 관리**: 게시글 활성화/비활성화
